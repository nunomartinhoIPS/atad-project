char** splitString ( char *  string,
int  nTokens,
const char *  delim 
)

Splits a string given a delimiter.

This function modifies the parameter 'string', replacing delimiter occurrences by '\0' and returns and array of pointers to each invidual tokens.

This array must be free'd by the caller.

Parameters
string[in] the string to split
nTokens[in] number of required tokens. If there are less tokens than specified, then NULL is used for subsequent tokens.
delim[in] delimiter, usually a single character
Returns
char** pointer array (of size nFields) to string "tokens".
